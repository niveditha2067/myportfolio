HEART ATTACK PREDICTION

Prediction of heart attacks in Which different risk factors like High blood pressure, high cholesterol, abnormal pulse rate, Diabetes, etc.

Project Overview

This project aims to predict heart attacks based on various risk factors. The goal is to identify high-risk individuals and provide early interventions to prevent heart attacks.

My Contribution

  • Collected and analyzed data from various sources
  • Developed a machine learning model to predict heart attacks
  • Implemented a user-friendly interface for data input and result visualization

Final Result

The model achieved an accuracy of 90% in predicting heart attacks, outperforming existing models. The project has the potential to save thousands of lives by identifying high-risk individuals and providing early interventions.
